Introduction

    For developers who wish to build their own StimulGL program or one of its default Plug-ins they can clone a git repository from https://github.com/svengijsen/StimulGL. This repository can be build/used/distributed under the terms of the GNU General Public License (GPL version 3), see http://www.gnu.org/licenses/gpl.html

    The StimulGL repository

    The repository contains the following directories structure:

    /Source directory:
    Here you can find all the StimulGL source code together with the Project/Solution files. Inside the /Source directory you find a /Plugin and /StimulGL folder. The /Plugin folder contains all the Plugin projects that can be used by the StimulGL main program. The source code for the StimulGL program can be found in the /StimulGL folder.

    /Install directory:
    This directory contains all the needed files and binaries that you need for a valid installation. Some of them first need to be build by your IDE and some of them are already available in the /Install directory. A post build can copy the build output from the IDE to this folder.

    /InstallJammer directory: This directory is the root directory for the InstallJammer project files and generated files. InstallJammer is used for building the various installers for the StimulGL application.

    Build Tools

    You can build your own version of StimulGL using Qt Creator or Microsoft Visual Studio 10.0. Both solution/project files can be found inside the repository. StimulGL can be build in the future for various platforms (Windows/Linux/Mac) with 32bit or 64bit support, at this moment only Windows 32 and 64-bit platforms are supported. StimulGL development will always try to stay closely in sync with the latest version of the Qt library. Only plugins that are build using the same version of the Qt library are compatible.

    Development Environment Setup

    The following guide shows how to install and configure a successful installation for VS2010 and Qt4.8.0 that is ready for building your own version of StimulGL.

    Visual Studio 2010 (SP1)

    • Install Visual Studio 2010 (32+64bit)
    • Install Visual Studio 2010 SP1
    • Install the Microsoft Windows SDK for Windows 7 and .NET Framework 4, make sure to select the Debugging Tools option under the Common Utilities and to select the Debugging Tools option under the Redistributable Packages. see http://www.microsoft.com/download/en/details.aspx?displaylang=en&id=8279

    Qt library for Windows

    Building the win32 Qt library:

    • Download and run the Qt libraries 4.8.0 for Windows (VS 2010) from http://qt.nokia.com/downloads and install in C:\Qt\4.8.0\win32, remaining installation options may be kept default

    Building the x64 Qt library:

    • Copy/Paste and Rename the folder “C:\Qt\4.8.0\win32” to “C:\Qt\4.8.0\x64”
    • Rename the file syncqt.bat to syncqt.bat_ in directory C:\Qt\4.8.0\x64\bin
    • Rename the file syncqt to syncqt_ in directory C:\Qt\4.8.0\x64\bin
    • Open the VS2010 X64 Win64 Command prompt
      • cd C:\Qt\4.8.0\x64
      • configure -qt-libpng -qt-libmng -qt-libtiff -qt-libjpeg
      • nmake
      • nmake install

    Qt Creator for Windows

    • Download and run the Qt Creator 2.4.0 for Windows from http://qt.nokia.com/downloads and install in C:\Qt\qtcreator-2.4.0
    • Open Qt Creator
      • Goto menu Tools->Options->Build&run->Qt Versions->Add
      • Create the target "Qt 4.8.0 (win32)", C:\Qt\4.8.0\win32\bin\qmake.exe
      • Create the target "Qt 4.8.0 (x64)", C:\Qt\4.8.0\x64\bin\qmake.exe
    • Close and open Qt Creator again
      • Open an example *.pro file and set file association (*.pro) to be opened with Qt Creator
      • Build the (win32) project and verify that it successfully build
      • Build the (x64) project and verify that it successfully build

    Qt Visual Studio Plugin

    • Install the Qt Visual Studio Plugin from http://qt.nokia.com/downloads (does not work with the Visual Studio Express edition!)
    • Open Visual Studio 10.0
      • Goto the menu Qt -> Qt Options -> Add Qt versions:
        • 4.8.0 (32bit) -> C:\Qt\4.8.0\win32
        • 4.8.0 (64bit) -> C:\Qt\4.8.0\x64 Close Visual Studio

    Additional StimulGL Headers and Libraries

    • Download and extract the additional StimulGL files
    • Merge from this archive the folder "4.8.0" with "C:\Qt\4.8.0\" (Adds all needed additional headers and libraries)

    Environment Variabeles

    • Add the string ;C:\Qt\4.8.0\win32\bin;C:\Qt\4.8.0\x64\bin;C:\Qt\4.8.0\win32\lib;C:\Qt\4.8.0\x64\lib to the end of the System PATH environment variable

    • Download and install TortoiseGIT from http://code.google.com/p/tortoisegit/downloads/list
