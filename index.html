Welcome

StimulGL v is a free/open source application.

It's optimized for creating flexible behavioral experiments supported by the script engine and plugin framework.

The application is currently only supported under Windows (32 and 64-bit architectures) but in the future StimulGL will be become cross-platform (Mac, Linux).
