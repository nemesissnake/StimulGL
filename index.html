Welcome

StimulGL v is a free/open source application.

StimulGL is a rich multiple document type editor that has code editing features like automatic code completing and syntax highlighting.

StimulGL contains a automatic Plug-in loading framework that fully supports devellopers to easily write various extensions.

StimulGL contains an internal script engine whose environement can access a large part of the Qt SDK and loaded Plug-ins by making use of script bindings.

StimulGL is optimized for creating and running flexible behavioral experiments supported by the script engine and plugin framework.

The application is currently only supported under Windows (32 and 64-bit architectures) but in the future StimulGL will be become cross-platform (Mac, Linux).
