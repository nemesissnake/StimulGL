Examples and Tutorials

In the StimulGL\examples\Tutorials\ directory you can find several sub-directories containing specific examples about some feature's of StimulGL. The StimulGL\examples\Tutorials\99_PluginExamples\ directory contains examples about how to use a specific Plug-in script class.

The StimulGL\examples\Tutorials\Common\ directory contains only files that are shared by the various examples just to save disk space used by the StimulGL installer. If you plan to create your own experiment or script using one of the examples, you could copy one of the sub-directories and use it as a template starting point but please bear in mind that there might be some references to the \Common\ sub-directory that you should not forget to copy.

The following sub-directories containing the various examples are present in the StimulGL\examples\Tutorials\ directory:

  • 1_MediaPlayer: this directory contains a Experiment that shows how you can embed and use Audio and Movie content in your experiments using QtQuick2.
  • 2_RetinoTopicMapping: this directory contains some Retinotopic example experiments.
  • 3_ImageBuffer: this directory contains a Experiment that show how you can make use of the ImageBuffer using QtQuick2 for quick image loading/buffering (in memory).
  • 4_UserInterfaces: this directory contains several script showing how you can create custom dialogs/User Interfaces containing several UI component.
  • 5_ScalableVectorGraphics: this directory contains some SVG (*.svg, Scalable Vector Graphics) that can be viewed dierectly inside StimulGL.
  • 6_RetinoTopicMappingOutput: this directory contains example files for running an Retinotopic Polar Angle experiment that can generate visual stimulation output. There are also some script that show an example of how a user can work with this output. Please read this description and tutorial if you're interested.

The following sub-directories containing the Plug-in script class examples are present in the StimulGL\examples\Tutorials\99_PluginExamples\ directory:

  • ExperimentManager
  • FirebirdClient
  • KeyBoardDevice
  • ParallelPortDevice
  • PiezoStimDevice
  • SerialPortDevice
  • TBVExchanger
  • TCPNetworkServer
  • USBHIDDevice
