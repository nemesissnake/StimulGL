Detailed Description

The Experiment Manager class.

The Experiment Manager can be used configure, open and execute experiment (*.exml) files.

bool ExperimentManager::loadExperiment ( QString  strSource = "",
bool  bViewEditTree = true,
bool  bIsFile = true 
) [slot]

Loads the Experiment from a file into memory.

The Experiment source (strSource) is load into memory (filepath or content), if strSource=="" then the last set experiment file is automatically loaded in memory, see setExperimentFileName. If bIsFile is set to true then the strSource contains a path to the experiment file, if bIsFile is set to false then the strSource contains the content itself. If bViewEditTree is set to true then the experiment filename is shown in a custom viewer (under construction).
