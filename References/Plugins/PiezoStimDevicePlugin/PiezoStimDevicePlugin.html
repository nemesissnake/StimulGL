Introduction

    This PiezoStim Device Plug-in provides all funtionality for accessing the QuaroSys Piezostimulator API.

    The QS Piezostimulator is a mechanical tactile stimulator, specified for use in functional imaging experiments like MEG, fMRI or EEG. The stimulation is realized by using several different pin matrices (2x4, 4x4, 2x5, 4x5, etc.). Each of the pins can be driven out to 4096 different heights between 0 and 1.5 mm with a timing accuracy of 0.5 ms.

    There are separate ports for input and output of trigger signals, enabling a precise synchronization between stimulation and measurement. For more complex experiments, it is even possible to upgrade the system.

    Plug-in script access

    An object of the PiezoStim Device Plug-in can be created using the following script command:

    var PiezoStimDeviceobject = new PiezoStimDevice();

    You can use this object to access all functions provided by the PiezoStim Device Plug-in. A list of all available functions can be found in the doxygen's reference documentation.
