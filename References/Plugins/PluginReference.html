Plugin Reference

StimulGL supports the use of Plug-ins to extend its functionality. The Plug-ins may extend the internal Script engine (Script Plugins) or the QML/QtQuick2 engine (QML Plugins).

Script Plugins

The script can be extended by two type of Plug-ins, namely Device or Extension Plug-ins. If your planning to extend StimulGL's functionality by writing your custom Script Plugin you might want to take a look at the ExtensionPlugin Template documentation which explains how to get started writing your own custom StimulGL Script Plug-in.

Below you can see a list of Script Plug-ins (Device and Extension) that are by default made available by the StimulGL installer.

Device Plugins

  • KeyBoardDevicePlugin
  • ParallelPortDevice
  • PiezoStimDevicePlugin
  • SerialPortDevicePlugin
  • USBHIDDevice

Extension Plugins

  • ExperimentManager
  • FireBirdClientPlugin
  • TBVExchanger
  • TCPNetworkServerPlugin

 

QML Plugins

  • StimulGL_QMLExtensions
