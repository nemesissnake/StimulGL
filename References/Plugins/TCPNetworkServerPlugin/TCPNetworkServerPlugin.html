Introduction

    This TCP Network Server Plug-in provides all kind of funtionality for creating and using a TCP socket for data transmission over a TCP/IP network. The plugin can be used for configuring either a network client or server object.

    Plug-in script access

    An object of the TCP Network Server Plug-in can be created using the following script command:

    var objTCPNetwork = new TCPNetworkServer();

    You can use this object to access all functions provided by the TCP Network Server Plug-in. A list of all available functions can be found in the doxygen's reference documentation.
