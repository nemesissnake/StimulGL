Introduction

    This Plug-in provides various features for capturing Keyboard events (a user interaction with the keyboard).

    Key events are sent to the widget with keyboard input focus when keys are pressed or released. These events can be automatically routed to an custom script handler function for further processing.

    Plug-in script access

    An object of the Keyboard Device Plug-in can be created using the following script command:

    var KeyBoardCaptureObj = new KeyBoardCapture();

    You can use this object to access all functions provided by the Keyboard Device Plug-in. A list of all available functions can be found in the Doxygen's reference documentation.
